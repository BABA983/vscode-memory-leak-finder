# VSCode Memory Leak Finder

Find memory leaks in vscode to improve robustness and performance

## Quickstart

```sh
git clone git@github.com:SimonSiefke/vscode-memory-leak-finder.git &&
cd vscode-memory-leak-finder &&
npm ci &&
npm run e2e
```
